Opdracht 1

Als je op de button hieronder klikt, dan komt er een prompt tevoorschijn die je vraagt om een kleur in te typen. De mogelijke kleuren zijn:

  • Rood
  • Groen
  • Blauw

Als de gebruiker een kleur heeft ingetypt, moet de naam van de kleur hieronder zichtbaar zijn. Gebruik if/else voor elke kleur. Bij alle andere kleuren toon je "onbekende kleur". Denk goed na wat je allemaal nodig hebt. Schrijf het op papier en voer het in JavaScript uit. Let op, schrijf de code in de file opdr1.js
